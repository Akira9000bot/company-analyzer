# Cost tracking: prices.json

Keys in `prices.json` must match your OpenClaw **model id** (`agents.defaults.model.primary`), e.g. `provider/model-id`.

- **input**: cost per 1M input tokens (USD)
- **output**: cost per 1M output tokens (USD)

Example entries are provided. Add or edit an entry for your configured LLM so cost tracking uses the correct rates.
